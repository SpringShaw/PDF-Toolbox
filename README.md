# PDF 工具箱

一个基于 Python 的本地 PDF 处理工具，无需联网，所有操作都在本地完成，保护隐私安全。

## 功能特性

- ✅ **PDF 合并** - 多个 PDF 文件按顺序合并成一个
- ✅ **PDF 拆分** - 将一个 PDF 拆分成多个文件
- ✅ **提取页面** - 从 PDF 中提取指定页面
- ✅ **页面旋转** - 旋转 PDF 中的页面方向
- ✅ **PDF 加密** - 为 PDF 设置打开密码
- ✅ **PDF 解密** - 移除已知密码的 PDF 保护
- ✅ **添加文本水印** - 自定义文字、位置、颜色、旋转角度

## 运行方式

### 方式一：直接运行 exe（推荐）

下载 `dist/` 目录下的 exe 文件，双击即可运行，无需安装 Python 环境。

### 方式二：Python 源码运行

1. 安装依赖：
```bash
pip install PyPDF2 pymupdf
```

2. 运行主程序：
```bash
python pdf_toolbox.py
```

## 使用说明

### PDF 合并
1. 选择 "PDF 合并"
2. 点击 "添加文件"，选择要合并的多个 PDF
3. 可调整文件顺序（上移/下移）
4. 选择输出文件路径
5. 点击 "执行操作"

### PDF 拆分
1. 选择 "PDF 拆分"
2. 选择要拆分的 PDF 文件
3. 选择拆分方式：
   - 按每页拆分（每页一个文件）
   - 按自定义范围拆分（如：1-3,5,7-10）
4. 选择输出文件夹
5. 点击 "执行操作"

### 提取页面
1. 选择 "提取页面"
2. 选择源 PDF 文件
3. 输入要提取的页码范围（如：1-3,5,7-10）
4. 选择输出文件路径
5. 点击 "执行操作"

### 页面旋转
1. 选择 "页面旋转"
2. 选择源 PDF 文件
3. 选择旋转角度（90°、180°、270°）
4. 选择要旋转的页码（留空则全部旋转）
5. 选择输出文件路径
6. 点击 "执行操作"

### PDF 加密
1. 选择 "PDF 加密"
2. 选择要加密的 PDF 文件
3. 输入密码并确认
4. 选择输出文件路径
5. 点击 "执行操作"

### PDF 解密
1. 选择 "PDF 解密"
2. 选择加密的 PDF 文件
3. 输入正确密码
4. 选择输出文件路径
5. 点击 "执行操作"

### 添加文本水印
1. 选择 "添加文本（水印）"
2. 选择要编辑的 PDF 文件
3. 输入水印文本内容（默认：机密）
4. 设置参数：
   - 应用页码（留空则全部页面）
   - 字体大小
   - 文本颜色
   - 旋转角度
5. 选择输出文件路径
6. 点击 "执行操作"

## 技术栈

- **GUI**: Python tkinter
- **PDF 处理**: PyPDF2
- **文本编辑**: PyMuPDF (fitz)

## 注意事项

- 所有操作均在本地完成，文件不会上传到任何服务器
- 请确保源 PDF 文件未损坏或被 DRM 保护
- 密码加密后请妥善保管，忘记密码无法恢复
