# PDF 工具箱

[English](./README.en.md) | **简体中文**

一个**完全离线**的 PDF 处理 Web 工具，无需安装、无需部署。集成 **PDF 预览、合并、拆分、提取、压缩、水印、删除页面、旋转、图片转PDF** 等常用功能，所有操作都在本地完成，保护隐私安全。

## 功能特性

### 🔀 PDF 合并
- 选择多个 PDF 文件，按顺序合并成一个

### ✂️ PDF 拆分
- 每页拆分为独立文件
- 按自定义页码范围拆分

### 📄 提取页面
- 从 PDF 中提取指定页码，生成新文件

### 🗜️ PDF 压缩
- 低 / 中 / 高三档质量可选
- 适合扫描件、图片型 PDF 的瘦身

### 🖼️ 图片转 PDF
- 支持 PNG、JPEG、WebP 格式
- 每张图片生成一页 PDF

### 👁️ PDF 预览
- 直接在浏览器中嵌入查看器预览 PDF

### ❌ 删除页面
- 按页码指定要删除的页面

### 🔄 页面旋转
- 支持 90°、180°、270° 旋转
- 可选指定页面或全部页面

### 💧 添加水印
- 自定义水印文本、字体大小、颜色
- 支持旋转角度和页码范围

## 使用方式

直接用浏览器打开 `index.html` 即可，无需安装、无需部署、完全离线。

```bash
# 方式一：双击 index.html 打开
# 方式二：本地服务器（推荐，可避免跨域问题）
python -m http.server 8080
# 访问 http://localhost:8080
```

## 技术栈

- **HTML5 + CSS3 + JavaScript** (原生，无框架依赖)
- **pdf-lib** — PDF 创建与编辑
- **pdf.js** — PDF 渲染与预览
- **Canvas** — 图片降质压缩

## 注意事项

- 所有操作均在本地完成，**文件不会上传到任何服务器**
- 建议使用 Chrome / Edge 最新版本以获得最佳体验
- 压缩功能对扫描型 PDF 效果显著，文字型 PDF 建议使用后端方案
